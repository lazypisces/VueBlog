# 券商與分類
  
  ![](../../.vuepress/public/images/基本功能_券商與分類.jpg)

 - **股票分類** : 
   - 除了可以同時設定不同的券商外，也能透過此功能，將股票進行分類
 - 手續費折數
   - 數字1為沒有折扣， **打8折** 請輸入 `0.8` 以此類推...
 - 零股最低手續費
   - 同最低手續費，但零股交易手續費經常低於20元，因此各券商在零股部分最低手續費有可能是1元
 - ⭐融資券 : 
   - 融資券的成本費用，各券商也有所不同。皆在此設定
   - 參考 [進階功能 > 功能 > 融資券](../PayOnly/進階功能.md#融資券)
