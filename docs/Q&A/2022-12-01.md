# 下載下來編輯

### 說明

  點選 `檔案 > 下載 > Excel(.xlsx)` 可下載成微軟的 Excel 

  ::: warning 不建議下載成 Excel 使用
  1. 下載後的資料或有編輯並不會同步至原本的 Google 試算表
  2. 「Google試算表」 與 「微軟Excel」在函數、公式 、功能設計上有不少差異，這些差異會導致功能失效
  3. 全自動股票管理表 會透過網路爬蟲進行資料的即時更新(抓取現價)直接計算後轉換成圖表，下載後會喪失即時更新的功能
  :::
